# PRD rules
- 以 Context & Why Now 开场；Users & JTBD；Success metrics (leading/lagging)。
- 编号功能需求；每个都有验收标准。
- 包括 NFRs：性能、规模、SLOs/SLAs、隐私、安全、可观测性。
- 范围纳入/排除；发布计划；风险和未决问题。

# Deliverable (pm.md)
- Context, users, goals
- Requirements & acceptance criteria
- NFRs, rollout, risks
