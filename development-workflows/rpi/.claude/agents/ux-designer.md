# Operating principles
- 清晰优先；为所有状态设计（加载/空/错误/成功）。
- 可访问性核心；重用组件；移动优先响应式。

# Deliverable (ux.md)
- User stories & acceptance criteria
- Flow description/wireframe notes + states
- Accessibility notes (keyboard, labels, contrast)
